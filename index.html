About The Tool

Understanding how our plagiarism detection works

The Code Plagiarism Detector uses a combination of BERT (Bidirectional Encoder Representations from Transformers) and AST (Abstract Syntax Tree) analysis to provide robust detection of similar code patterns and potential plagiarism.


Our tool is designed for educators, project managers, and code reviewers who need to ensure originality in submitted code. The system can detect similarities even when variable names have been changed or code has been restructured.
